# Student Weekly Budget Tracker

## Description
This Python program helps students track their weekly expenses.  
The user enters their name, weekly budget, and up to 4 expense entries.  
Each expense is categorized and checked if it exceeds 25% of the budget.  
At the end, the program shows total spending, remaining budget, and budget status.

## Files
midterm_solution.py — main program file
README.md — project documentation

## How to Run
1. Make sure Python is installed on your computer.
2. Save the file as `midterm_solution.py`.
3. Open a terminal or command prompt.
4. Run the program using: python midterm_solution.py
